About my site

What I have learned

In this practical web design and coding experience, I have gained more about design thinking and the ability to fix issues and more about HTML, CSS, and even Javascript.

At first, I didn't know how to choose a suitable colour scheme for my theme, and then I found some websites that could help with colour choices after searching the internet. In addition, I collected some valuable websites that supply multiple icons and pictures.

To meet more needed features, I found the tutorials such as centre items with div tags, dynamically displaying text and images, responsively changing the size, clicking to switch pictures, etc.

The following are some tips or tricks that I summarised.

Tips&Tricks

  1. Design sketch before coding.

    Suppose I designed my sketch on paper beforehand, not just having a rough layout idea and coding. I will save a lot of time trying and adjusting the layout.

  2. Keep the layout, colour scheme and typography consistent.

    When I programmed the website, I kept the same structure, colour scheme and typography on all pages. It helps me hugely with page consistency.

  3. Unified the naming format.

    With the help of unified naming, all files, modules, classes and comments. I organised, modified and spotted errors easier.

  4. Avoid cumbersome colour schemes.

    In the beginning, My website looks monotonous and not eye-catching. So I refer to the professional colour-collection website to choose colours. However, the overall style of the colour scheme is not what I want after comparing and adjusting the colour many times.

  5. Use responsive design.

    At first, I thought the responsive design was the content automatically changing with the window size. However, when implementing the code, I found that there should be a reasonable range for size rather than adjusting the size without thinking.

  6. Stay inspired by others.

    When I always code and design myself, I can not find a satisfactory theme and good colour matching. However, with the help of browsing other websites and design platforms, I found a suitable style pattern I should follow.

Some improvement

After getting feedback on assignment 1, I redesigned the website's whole style to be simpler, concise, attractive, and consistent on all other pages with responsible size.

In addition, I combined various CSS files into a whole one with more discriminated classes and id names. I then modified similar layouts on different pages into the same Class type.
